## 1. ເພີ່ງກ່ຽວກັບແອັບ

**OmniNexus** ເປັນແອັບພລິເຄຊັນລວມທີ່ປະກອບໄປດ້ວຍເຄື່ອງມື ແລະ ຂໍ້ມູນຈຳນວນຫຼາຍ ທີ່ຖືກອອກແບບມາເພື່ອ:

- **ຄົນຕາບອດ ແລະ ຜູ້ພິການທາງສາຍຕາ** — ຮອງຮັບ TalkBack/VoiceOver, ສັ່ງງານດ້ວຍສຽງ, ຟັງຂໍ້ມູນຜ່ານ TTS
- **ຜູ້ສູງອາຍຸ ແລະ ຜູ້ມີສາຍເລືອນຮາງ** — ໂໝດພິມໃຫຍ່, High Contrast, ແວ່ນຂະຫຍາຍ, ໄຟສາຍ
- **ຜູ້ໃຊ້ທົ່ວໄປ** — ຂໍ້ມູນດາລາສາດ, ສະພາບອາກາດ, ຂ່າວດິນໄຫວ, ສູດອາຫານ, ແປງສະກຸນເງິນ ແລະ ອີກຫຼາຍຢ່າງ

**ພາສາ UI ຫຼັກ:** ພາສາລາວ (ຊື່ຟີເຈີ, ປຸ່ມ, ຄຳອະທິບາຍ)  
**ຂໍ້ມູນພາຍໃນ:** ອັງກິດ (API, Database, Wikipedia extracts, ML Kit labels)

## 2. ໜ້າຫຼັກ (Dashboard)

ເມື່ອເປີດແອັບຂຶ້ນມາ ທ່ານຈະເຫັນ **ໜ້າຫຼັກ (Dashboard)** ເຊິ່ງປະກອບດ້ວຍ:

### 2.1 ສ່ວນຫົວ (Header)
- **ໂລໂກ້ OmniNexus** + ຊື່ແອັບ
- **ໄອຄອນສະພາບອາກາດ** — ສະແດງຊຸດສີປັດຈຸບັນ (☀️/☁️/🌧️/🌙)
- ກົດ theme indicator ເພື່ອເປີດ **Theme Picker**

### 2.2 ເມນູບໍລິການ (ຈັດກຸ່ມ 5 ໝວດ)

#### ກຸ່ມ 1: ອະວະກາດ ແລະ ຂໍ້ມູນໂລກ
1. **ຮູບດາວເຄາະ (NASA APOD)** — ຮູບພາບດາລາສາດປະຈຳວັນ
2. **ຕິດຕາມ ISS** — ແຜນທີ່ສະຖານີອາວະກາດສາກົນແບບລຽວທາມ
3. **Global Dashboard** — ສະຖິຕິໂລກລຽວທາມ

#### ກຸ່ມ 2: ສະພາບອາກາດ ແລະ ສິ່ງແວດລ້ອມ
4. **ພະຍາກອນອາກາດ** — ພະຍາກອນ 7 ວັນ
5. **ຄຸນນະພາບອາກາດ** — AQI, PM2.5, PM10
6. **ແຜ່ນດິນໄຫວ** — ແຈ້ງເຕືອນຈາກ USGS

#### ກຸ່ມ 3: ສຸຂະພາບ ແລະ ຄວາມປອດໄພ
7. **ສຸຂະພາບຈິດ** — BMI/BMR, ແບບສອບຖາມສຸຂະພາບຈິດ
8. **ເຫດສຸກເສີນ** — SOS, ປະຖົມພະຍາບານ, ແຈ້ງເຕືອນກິນຢາ
9. **ໂຣກລະບາດ** — COVID-19 + ວັກຊີນ + Travel Check
10. **ການເຂົ້າເຖິງ** — High Contrast, ຂະໜາດຕົວອັກສອນ, Reduce Motion

#### ກຸ່ມ 4: ອາຫານ ແລະ ຊີວິດ
11. **ສູດອາຫານ** — ຄົ້ນຫາສູດຈາກສ່ວນປະກອບ (Edamam API)
12. **ສະແກນດ້ວຍ AI** — ຖ່າຍຮູບວັດຖຸດິບ → ML Kit ຈຳແນກ → ຮູບ + ຂໍ້ມູນ Wikipedia
13. **ຄັດແຍກຂີ້ເຫຍື້ອ** — ຖານຂໍ້ມູນຫຼາຍກວ່າ 70 ລາຍການ + ສະແກນດ້ວຍ AI + Green Points

#### ກຸ່ມ 5: ເງິນ ແລະ ການເງິນ
14. **ຂໍ້ມູນປະເທດ** — REST Countries API, ແຜນທີ່, ປຽບທຽບ
15. **ເວລາໂລກ** — World Time + ເຂດເວລາ
16. **ແປງສະກຸນເງິນ** — 40+ ສະກຸນເງິນ, ອັດຕາແລກປ່ຽນລຽວທາມ
17. **ຫຸ້ນ ແລະ Crypto** — ຕິດຕາມລາຄາ + ພອດຕະຟອລິໂອ

#### ກຸ່ມ 6: ການເຂົ້າເຖິງ ແລະ ຄວາມປອດໄພ (ເພີ່ມໃໝ່)
18. **SafeWay ນຳທາງ** — GPS ນຳທາງດ້ວຍສຽງຈຸດຕໍ່ຈຸດ
19. **Super Magnifier** — ແວ່ນຂະຫຍາຍກ້ອງ + ໄຟສາຍ + ປັບ Contrast
20. **Emergency Hub** — SOS, Shake for Help, Voice Directory

### 2.3 ສ່ວນລຸ່ມ
- **ແຈ້ງເຕືອນດື່ມນ້ຳ** — ເປີດ/ປິດການແຈ້ງເຕືອນທຸກ 1 ຊົ່ວໂມງ
- **Voice Commands** — ສະແດງສະຖານະການຮັບຟັງ ຫຼື ກຳລັງເວົ້າ
  - ປຸ່ມ Play ▶️ — ຟັງ Briefing ລາຍການບໍລິການ
  - ປຸ່ມ Mic 🎤 — ເລີ່ມຟັງສຽງສັ່ງ
  - ປຸ່ມ Stop ⏹️ — ຢຸດການຟັງ/ການເວົ້າ

## 4. ການປ່ຽນຊຸດສີ (Theme)

### 4.1 ຊຸດສີທີ່ມີ

| ຊື່ | ໄອຄອນ | ລັກສະນະ |
|---|---|---|
| Sunny ☀️ | ຮູບດວງຕາເວັນ | Gradient ອຸ່ນ (ສົ້ມ-ເຫຼືອງ) + ຄີງຕາເວັນ |
| Cloudy ☁️ | ຮູບເມກ | Gradient ມຸດ (ຟ້າ-ເທົາ) |
| Rainy 🌧️ | ຮູບນ້ຳຝົນ | Gradient ເຢັນ (ຟ້າ-ມ່ວງ) + ຝົນຕົກ |
| Night 🌙 | ຮູບພະຈັນ | Gradient ມືດ (ດຳ-ຟ້າ) + ດາວກະພິບ |
| **High Contrast Mode** | ໄອຄອນ Contrast | ສີດຳ-ຂາວ, ຂອບເຂັ້ມ, ໃຊ້ງານກັບ TalkBack/VoiceOver ໄດ້ດີ |

### 4.2 ວິທີປ່ຽນ
- ກົດທີ່ **ຊື່ຊຸດສີ** (ຢູ່ມຸມຂວາບນບຂອງໜ້າຫຼັກ)
- ຫຼື ກົດທີ່ **ໄອຄອນສະພາບອາກາດ** ໃນ AppBar
- ເລືອກຊຸດສີທີ່ຕ້ອງການຈາກ Bottom Sheet

## 5. ລາຍການຟີເຈີທັງໝົດ — ຄຳອະທິບາຍລະອຽດ

---

### 5.1 ຮູບດາວເຄາະ — NASA APOD (`apod_page.dart`)

**ລາຍລະອຽດ:**  
ດຶງຮູບພາບດາລາສາດປະຈຳວັນຈາກ NASA (Astronomy Picture of the Day)

**ການໃຊ້ງານ:**
- ເມື່ອເຂົ້າມາ ຈະໂຫຼດຮູບປະຈຳວັນລ່າສຸດໂດຍອັດຕະໂນມັດ
- ກົດຮູບເພື່ອເບິ່ງຂະຫຍາຍ
- ເລື່ອນລົງເພື່ອອ່ານຄຳອະທິບາຍ (ອັງກິດ)
- ກົດປຸ່ມວັນທີ (Date Picker) ເພື່ອເລືອກເບິ່ງຮູບຍ້ອນຫຼັງ
- ປຸ່ມ Share — ແບ່ງປັນຮູບ ແລະ ລາຍລະອຽດ
- ປຸ່ມ Volume — ຟັງຄຳອະທິບາຍຜ່ານ TTS

**API:** NASA APOD API (ຟຣີ)

---

### 5.2 ຕິດຕາມ ISS — ISS Tracker (`iss_page.dart`)

**ລາຍລະອຽດ:**  
ແຜນທີ່ສະແດງຕຳແໜ່ງແບບລຽວທາມຂອງສະຖານີອາວະກາດສາກົນ (ISS) ພ້ອມຕຳແໜ່ງ GPS ຂອງຜູ້ໃຊ້

**ການໃຊ້ງານ:**
- ກົດປຸ່ມ **"ISS Tracker"** ຈາກ Dashboard
- ແຜນທີ່ຈະສະແດງ:
  - **ຈຸດສີຟ້າ** — ຕຳແໜ່ງປັດຈຸບັນຂອງຜູ້ໃຊ້
  - **ໄອຄອນ ISS** — ຕຳແໜ່ງປັດຈຸບັນຂອງສະຖານີອາວະກາດ
  - **ເສັ້ນທາງໂຄຈອນ** — ເສັ້ນສະແດງທາງໂຄຈອນ
- ດ້ານລຸ່ມມີຂໍ້ມູນເພີ່ມເຕີມ:
  - ໄລຍະຫ່າງຈາກຜູ້ໃຊ້
  - ຄວາມໄວ, ລະດັບຄວາມສູງ
  - ເວລາທີ່ ISS ຈະຜ່ານໃກ້ຜູ້ໃຊ້
- ກົດປຸ່ມ **"Listen"** ເພື່ອໃຫ້ TTS ອ່ານຂໍ້ມູນ

**API:** Open-Notify ISS API + Geolocator GPS

### 5.12 ສະແກນດ້ວຍ AI — Smart Lens (`smart_lens_page.dart`)

**ລາຍລະອຽດ:**

**ການໃຊ້ງານ:**
1. ກົດ **"ຖ່າຍຮູບ"** ຫຼື **"ເລືອກຈາກຄັງຮູບ"**
2. ML Kit ຈະວິເຄາະ ແລະ ສະແດງລາຍການວັດຖຸທີ່ກວດພົບ
3. ແຕ່ລະລາຍການສະແດງ: ຊື່ (ລາວ + ອັງກິດ) + ຄວາມແນ່ນອນ (%)
4. **ກົດທີ່ຊື່ວັດຖຸ** → ເປີດໜ້າ **`Image Search`**
   - ດຶງຮູບຕົວຢ່າງ + ຄຳອະທິບາຍຈາກ Wikipedia API
   - ກົດ "ເບິ່ງເພີ່ມເຕີມ" → ເປີດ Wikipedia ໃນ Browser

**ແຜນທີ່ແປຊື່:** ມີ dictionary 50+ ຄຳ (food→ອາຫານ, fruit→ໝາກໄມ້...)

**ການໃຊ້ງານ — ແຖບ 3: ຄະແນນ (Points)**

ລະບົບນຳທາງດ້ວຍສຽງແບບຈຸດຕໍ່ຈຸດ ສຳລັບຄົນຕາບອດ ໃຊ້ GPS + TTS

**ການໃຊ້ງານ — ການຕັ້ງຄ່າ:**

1. **ການບັນທຶກຈຸດເຊັກພອຍ (Checkpoint)**
   - ເມື່ອທ່ານຢູ່ທີ່ຈຸດສຳຄັນ (ເຊັ່ນ: ໜ້າເຊວ່ນ, ປາກຊອຍ)
   - ກົດປຸ່ມ **FAB (+)** ມຸມຂວາລຸ່ມ
   - ພິມຊື່ຈຸດ ແລະ ຄຳບອກທາງ (ເຊັ່ນ: "ເຖິງໜ້າເຊວ່ນແລ້ວ, ໃຫ້ລ້ຽວຊ້າຍ")
   - ກົດ Save — ຈຸດຈະປາກົດເທິງແຜນທີ່

2. **ການໃຊ້ງານປະຈຳວັນ**
   - ເປີດ SafeWay → ແຜນທີ່ສະແດງຕຳແໜ່ງທ່ານ + ຈຸດ checkpoint ທັງໝົດ
   - ຍ່າງໄປຕາມເສັ້ນທາງ → ເມື່ອໃກ້ຈຸດ (20 ແມັດ) → TTS ຈະອ່ານຄຳບອກທາງໂດຍອັດຕະໂນມັດ
   - ຢູ່ດ້ານລຸ່ມມີສະຖານະ "Listening..." ພ້ອມຈຳນວນຈຸດ

3. **ການຈັດການຈຸດ**
   - ກົດທີ່ Marker ເທິງແຜນທີ່ → ເບິ່ງຄຳບອກທາງ + ກົດ Play ເພື່ອຟັງອີກຄັ້ງ
   - ກົດ Edit → ແກ້ໄຂຊື່/ຄຳບອກທາງ
   - ກົດ Delete → ລົບຈຸດ

**ເທັກໂນໂລຢີ:**
- **GPS:** Geolocator `getPositionStream(distanceFilter: 5)` — ອັບເດດທຸກ 5 ແມັດ
- **Geofencing:** ຄຳນວณໄລຍະຫ່າງຈາກ checkpoint ທຸກຄັ້ງທີ່ GPS ປ່ຽນ
- **TTS:** ລາວ (th-TH) / ອັງກິດ (fallback), Cooldown 30 ວິນາທີຕໍ່ຈຸດ
- **ບັນທຶກ:** Hive (`safe_way`) — ຂໍ້ມູນຢູ່ຄົງທົນເຖິງປິດແອັບ
- **ແຜນທີ່:** FlutterMap (OpenStreetMap tiles)

### 5.19 Super Magnifier (`magnifier_page.dart`) — **ໃໝ່!**

**ລາຍລະອຽດ:**  
ແວ່ນຂະຫຍາຍກ້ອງສຳລັບຄົນສາຍເລືອນຮາງ ພ້ອມໄຟສາຍ ແລະ ການປັບສີ

**ການໃຊ້ງານ:**

1. **ການຂະຫຍາຍ (Zoom)**
   - ໃຊ້ **Pinch-to-Zoom** (ຫຍິບສອງນິ້ວເພື່ອຂະຫຍາຍ/ຫຍໍ້)
   - ມີຕົວເລກສະແດງລະດັບ Zoom (ເຊັ່ນ: 2.5x)

2. **ການປັບຄຸນນະພາບຮູບ**
   - **Contrast Filter** — ກົດປຸ່ມ Contrast (🎨) ເພື່ອສະຫຼັບໂໝດ:
     - ປົກກະຕິ → ໂໝດຕົວໜັງສືເຫຼືອງ-ພື້ນຫຼັງດຳ (ອ່ານງ່າຍຂຶ້ນ)
   - **Brightness Slider** — ປັບຄວາມສະຫວ່າງຂອງໜ້າຈໍເພີ່ມເຕີມ

3. **ໄຟສາຍ (Flashlight)**
   - ກົດປຸ່ມ Flash 🔦 — ເປີດໄຟສາຍກ້ອງໃນທີ່ມືດ

4. **Freeze Frame (ຖ່າຍຮູບຄ້າງ)**
   - ກົດປຸ່ມ Freeze 📷 — ຖ່າຍຮູບຄ້າງໄວ້
   - ໃຊ້ນິ້ວຂະຫຍາຍຮູບ (InteractiveViewer) — ເບິ່ງລາຍລະອຽດໄດ້ດີ
   - ກົດ Live ກັບຄືນ — ກັບໄປເບິ່ງກ້ອງສົດ

5. **Grid Lines (ເສັ້ນຕາຕາລາງ)**
   - ກົດປຸ່ມ Grid (⊞) — ສະແດງ/ຊ່ອນ ເສັ້ນຕາຕາລາງ 3×3
   - ຊ່ວຍຈັດວາງຕຳແໜ່ງສິ່ງທີ່ຕ້ອງການເບິ່ງ

**ການຈັດວາງ UI:**
- ເຕັມຈໍ — ກ້ອງສົດ
- Bottom Bar 4 ປຸ່ມ: Flash, Contrast, Grid, Freeze
- Zoom indicator ລອຍເທິງ
- Brightness slider ຢູ່ລຸ່ມສຸດ

### 5.20 Emergency Hub (`emergency_hub_page.dart`) — **ໃໝ່!**

**ລາຍລະອຽດ:**  
ສູນລວມການຂໍຄວາມຊ່ວຍເຫຼືອສຸກເສີນສຳລັບຄົນຕາບອດ — SOS, Shake, Voice Directory

**ການໃຊ້ງານ:**

#### 5.20.1 SOS Big Button
- ປຸ່ມແດງໃຫຍ່ກວມ 40% ຂອງຈໍ
- **ກົດ 1 ຄັ້ງ:** TTS ຈະຮ້ອງ "SOS!" ດັງໆ + ສະແດງຕົວເລືອກ
- **ກົດຄ້າງ (Long press):** ສົ່ງ SMS ພິກັດ GPS ໄປຫາທຸກເບີໂທໃນລາຍຊື່ສຸກເສີນທັນທີ
  - SMS ມີລິ້ງ Google Maps: `https://maps.google.com/?q=ລະຕິຈູດ,ລອງຈິຈູດ`

#### 5.20.2 Shake for Help
- **ເຂຍ່າ (Shake) ມືຖືແຮງໆ** → ລະບົບກວດຈັບຜ່ານ Accelerometer (> 25 m/s²)
- ຈະມີການນັບຖອຍຫຼັງ 5 ວິນາທີ
- ຖ້າບໍ່ກົດຍົກເລີກ → ສົ່ງ SMS GPS ອັດຕະໂນມັດ
- ສະແດງສະຖານະ "Shake Detected!" ເທິງຈໍ

#### 5.20.3 Voice Directory
- ລາຍຊື່ຜູ້ຕິດຕໍ່ສຸກເສີນ ທີ່ມີປຸ່ມໃຫຍ່
- **ກົດທີ່ຊື່:** TTS ຈະອ່ານຊື່ (ເປັນລາວ)
- **ກົດທີ່ປຸ່ມໂທ (Phone icon):** ໂທອອກທັນທີ
- ສາມາດເພີ່ມຜູ້ຕິດຕໍ່ໄດ້ຜ່ານ Settings

**ຜູ້ຕິດຕໍ່ຕົ້ນສະບັບ (Sample contacts):**
| ຊື່ | ເບີ |
|---|---|
| ພໍ່ | 020 12345678 |
| ແມ່ | 020 87654321 |
| ອ້າຍ | 020 11223344 |
| ເອື້ອຍ | 020 55667788 |
| ຕຳຫຼວດ 191 | 191 |
| ໂຮງໝໍ 1669 | 1669 |

**ເທັກໂນໂລຢີ:**
- **Accelerometer:** sensors_plus — ກວດຈັບ Shake
- **SMS:** url_launcher (`sms:number?body=message`)
- **ໂທອອກ:** url_launcher (`tel:number`)
- **GPS:** Geolocator `getCurrentPosition()`
- **TTS:** flutter_tts ອ່ານຊື່ຜູ້ຕິດຕໍ່
- **ບັນທຶກ:** Hive (`emergency_contacts`)
